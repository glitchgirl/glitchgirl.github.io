I manage multiple ecommerce sites, using magento and woocommerce. I also run smaller, non ecommerce sites for information for the store. I run these sites from server to css. Notably, during my time I have over tripled the site speed and helped ensure their security. I want to continue my serious web dev skills, especially in a structured environment where I can continue to use my skills.
